# Weather App

A weather app that lets you search any city and view current weather conditions.

## Features
- Search weather by city name
- Displays temperature, feels like, humidity and wind speed
- Toggle between °C and °F
- Weather icon and dynamic background based on conditions

## Built With
- Vanilla JavaScript
- OpenWeatherMap API
- Vite
